# Zotero workflow

Zotero manages references, PDFs, notes, and shared libraries.

## Basic workflow

1. Save papers to Zotero.
2. Put them in the correct shared library or collection.
3. Clean metadata: title, authors, year, journal, DOI.
4. Use citation keys consistently if exporting BibTeX.
5. Export or sync `.bib` files used by LaTeX projects.

## Good habits

- Do not rely on random PDF filenames.
- Check metadata before citing.
- Keep shared libraries organized by project or topic.
- Prefer DOI-backed entries when available.
- Do not manually type bibliography entries unless there is no alternative.

## With Overleaf

Use a `.bib` file exported from Zotero or maintained in the paper repository. Keep the source of truth clear.
